# プロジェクト概要

## プロジェクト名
食品ロス削減アプリ (Food Waste Reduction App)

## 目的
食品の在庫管理から廃棄削減まで、包括的な食品ロス削減を支援するアプリケーションを開発する。MCPサーバーを活用したデータ管理と、AI機能による賢い提案機能が特徴。

## 主要機能
1. 冷蔵庫・食品在庫管理
   - 食品登録（手入力、バーコード、音声）
   - 在庫管理とカテゴリ別表示
2. 賞味期限アラート機能
   - 期限チェックと通知
3. 使い切りレシピ提案
   - AIによる食材活用レシピ提案
4. 買い物リスト自動生成
   - 不足食材の検出と外部データ連携
5. 食材保存方法アドバイス
   - 保存知識ベースとAIアドバイス
